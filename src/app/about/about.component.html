About Debugi

What is Debugi?

Debugi is a comprehensive collection of developer utilities designed to streamline your workflow. Whether you're debugging, transforming data, or working with various formats, Debugi provides the tools you need in one convenient location.

Available Tools

Calculator
CSV Formatter
String Utils
QR Code Generator
Date Utils
Number Utils
Array Utils
JSON Visualizer
JSON Query
Regex Tester
Performance Monitor

Features

  • Dark/Light theme support
  • Global search across all tools
  • Keyboard shortcuts for efficiency
  • Responsive design for all devices
  • Fast, client-side processing
  • Privacy-focused (no data sent to servers)

Keyboard Shortcuts

Ctrl + K Open global search
Ctrl + Shift + T Toggle theme
