Quick Reference

Basic Patterns

  • . - Any character except newline
  • \d - Any digit (0-9)
  • \w - Any word character (a-z, A-Z, 0-9, _)
  • \s - Any whitespace character
  • ^ - Start of string
  • $ - End of string

Quantifiers

  • * - Zero or more
  • + - One or more
  • ? - Zero or one
  • {{ '{' }}n{{ '}' }} - Exactly n times
  • {{ '{' }}n,{{ '}' }} - n or more times
  • {{ '{' }}n,m{{ '}' }} - Between n and m times

Character Classes

  • [abc] - Any of a, b, or c
  • [a-z] - Any lowercase letter
  • [A-Z] - Any uppercase letter
  • [0-9] - Any digit
  • [^abc] - Not a, b, or c
  • (a|b) - Either a or b
